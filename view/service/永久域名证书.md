## 一、教程说明
- 证书类型：**Let's Encrypt 泛域名免费证书**，支持根域名 + 所有二级域名
- 适用场景：阿里云购买的域名、Ubuntu 服务器、Nginx 服务、七牛云对象存储
- 核心优势：全程免费、自动续期、自动同步七牛云，无需手动维护
- 环境要求：Ubuntu 服务器、已安装 Nginx、域名已解析到服务器公网 IP

## 二、前置准备
### 1. 阿里云配置
1.  登录阿里云「RAM 访问控制」控制台：https://ram.console.aliyun.com/users
2.  新建 RAM 用户，勾选「Open API 调用访问」，创建后**保存 AccessKey ID 和 AccessKey Secret**（仅显示一次）
3.  为该用户添加权限：`AliyunDNSFullAccess`（云解析 DNS 全权限）

### 2. 七牛云配置
1.  登录七牛云控制台，进入「个人中心 - 密钥管理」
2.  **保存 AccessKey 和 SecretKey**，用于后续自动部署证书

### 3. 服务器准备
- 域名已完成解析，指向当前 Ubuntu 服务器公网 IP
- 服务器已安装 Nginx，80/443 端口已开放

## 三、服务器端操作（核心步骤）
### 1. 安装 acme.sh 工具
```bash
# 安装 acme.sh，替换为你的邮箱
curl https://get.acme.sh | sh -s email=your@email.com

# 刷新环境变量
source ~/.bashrc

# 切换默认 CA 为 Let's Encrypt
acme.sh --set-default-ca --server letsencrypt
```

### 2. 配置阿里云 DNS 认证
```bash
# 替换为你自己的阿里云 AccessKey
export Ali_Key="你的阿里云AccessKey ID"
export Ali_Secret="你的阿里云AccessKey Secret"
```

### 3. 申请泛域名 HTTPS 证书
```bash
# 替换为你的根域名，申请泛域名证书（支持 *.yourdomain.com 所有二级域名）
acme.sh --issue --dns dns_ali -d yourdomain.com -d *.yourdomain.com
```
出现 `Cert success` 即证书申请成功。

### 4. 安装证书到 Nginx 目录
```bash
# 创建证书存放目录
mkdir -p /etc/nginx/ssl/yourdomain.com

# 部署证书，配置自动重载 Nginx + 七牛云自动部署钩子
acme.sh --install-cert -d yourdomain.com \
--key-file /etc/nginx/ssl/yourdomain.com/key.pem \
--fullchain-file /etc/nginx/ssl/yourdomain.com/fullchain.pem \
--deploy-hook "qiniu" \
--reloadcmd "systemctl reload nginx"
```

### 5. 配置七牛云密钥，触发首次自动上传
```bash
# 替换为你自己的七牛云 AccessKey
export QINIU_AK="你的七牛云AccessKey"
export QINIU_AK="你的七牛云SecretKey"

# 手动触发一次证书上传到七牛云（仅需执行一次）
acme.sh --deploy -d yourdomain.com --deploy-hook qiniu
```
出现 `Certificate successfully uploaded` 即上传成功（`no such domain` 为正常提示，仅自动绑定域名失败，证书已上传）

## 四、Nginx HTTPS 配置
### 1. 编辑 Nginx 站点配置
```bash
nano /etc/nginx/sites-available/default
```

### 2. 写入 HTTPS 配置（替换为你的域名）
```nginx
# 80 端口强制跳转 HTTPS
server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;
    return 301 https://$host$request_uri;
}

# HTTPS 服务配置
server {
    listen 443 ssl;
    server_name yourdomain.com www.yourdomain.com;

    ssl_certificate /etc/nginx/ssl/yourdomain.com/fullchain.pem;
    ssl_certificate_key /etc/nginx/ssl/yourdomain.com/key.pem;

    # 安全加密配置
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    root /var/www/html;
    index index.html;
}
```

### 3. 保存并重启 Nginx
```bash
# 检查配置是否正确
nginx -t

# 重启 Nginx 生效
systemctl restart nginx
```

## 五、七牛云后台操作（最后一步）
1.  登录七牛云控制台，进入「SSL 证书管理」
2.  找到已自动上传的泛域名证书（通用名称：`yourdomain.com` / `*.yourdomain.com`）
3.  点击证书右侧「部署」按钮
4.  选择需要绑定的二级域名（如 `img.yourdomain.com`），确认部署
5.  完成后，该二级域名即可正常使用 HTTPS 访问

## 六、自动续期说明
acme.sh 会自动创建 cron 定时任务，无需手动维护：
- 查看定时任务：`crontab -l`
- 任务逻辑：每天自动检查证书有效期，到期前自动续期
- 续期后自动同步新证书到七牛云，服务器重启不影响任务执行
- 证书永久免费，全程无需人工操作
